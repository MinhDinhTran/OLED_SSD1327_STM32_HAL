# OLED 1.32" with SSD1327 with 16-gray scale.

Library is using STM32 HAL libs.

## Software and HAL libraries.

- STM32F401RE
- STM32CubeIDE 1.3.0
- STM32CubeMX 5.6.0 in IDE
- HAL F4 1.25
